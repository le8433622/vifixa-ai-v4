# 🚀 Vifixa AI-Native Platform v4

## Kiến trúc: 4 Super-Agents + 3 Screens

### 4 Super-Agents (thay thế 36 functions cũ)
```
supabase/functions/v4-core/         ← Shared types + Zod + cost estimation
supabase/functions/v4-orchestrator/ ← Điều phối: nhận đơn, chat, phân tích
supabase/functions/v4-navigator/    ← Bản đồ: tìm gần, OSRM, heatmap
supabase/functions/v4-monetizer/    ← Tiền: định giá, upsell, thương lượng
supabase/functions/v4-humanizer/    ← Tương tác: chat, chẩn đoán, coach
```

### 3 Screens (thay thế 54 screens cũ)
```
/v4           ← 🗺️ MAP — dùng chung Khách/Thợ/Admin
/v4/chat      ← 💬 CHAT — mọi tương tác qua AI
/v4/dashboard ← 📊 DASHBOARD — AI insights, không table
```

### Design principles
1. **AI-native**: Form là chat, table là map, AI quyết định mọi thứ
2. **Map-first**: Screen duy nhất cho mọi role, chỉ khác data
3. **Streaming chat**: Mọi thứ qua hội thoại, không click
4. **Real-time**: Dashboard toàn AI insights, không refresh
